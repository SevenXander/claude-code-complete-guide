SCENE 1 - 什么是 Agent Loop？
Agent Loop 是 AI Agent 的 心跳！
它让 LLM 不只是"回答问题"，而是能 调用工具、执行操作、再思考，
像一个真正的 智能助手 一样持续工作。
位置: query.ts:241 queryLoop() 函数

// queryLoop 核心伪代码 async function* queryLoop() { while (true) { // 1. 流式调用 LLM API for await (const msg of callModel({...})) yield msg // 实时返回给用户 // 2. 检测 tool_use blocks if (toolUseBlocks.length > 0) needsFollowUp = true // 3. 执行工具 & 收集结果 for await (const r of runTools(...)) yield r // 4. 决定是否继续 if (!needsFollowUp) break // 否则 → 带着工具结果继续循环! } }
