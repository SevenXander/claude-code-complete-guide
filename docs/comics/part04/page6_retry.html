SCENE 2 - 常见敌人（错误类型）
🚦
429 - 速率限制
请求太快了！服务器说：慢点来
→ 等一会儿再试
🔥
529 - 服务器过载
服务器忙不过来！最多重试 3 次
→ 只在前台任务重试
🔌
ECONNRESET / EPIPE
网络断了！像电话线被剪断
→ 重建连接并重试
