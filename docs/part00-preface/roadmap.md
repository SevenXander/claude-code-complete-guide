# 学习路线图

> 不管你是零基础小白还是资深开发者，都能找到适合自己的学习路径。

---

## 路线一：零基础入门（推荐新手）

如果你没有任何编程经验，按照这个顺序学习：

```
第一篇：基础入门（必读）
  │  学习目标：理解什么是 Claude Code、终端、TypeScript、React
  ▼
第二篇：架构全景（必读前3章）
  │  学习目标：建立 Claude Code 的整体认知
  ▼
第十二篇：亮点总结（快速浏览）
  │  学习目标：了解 Claude Code 的 10 大核心优势
  ▼
第三篇：工具系统（选读）
  │  学习目标：理解 AI 如何使用工具
  ▼
感兴趣的其他篇章
```

**预计时间**：2-3 周（每天 1-2 小时）

---

## 路线二：前端/全栈开发者

如果你有 JavaScript/TypeScript 开发经验：

```
第二篇：架构全景（必读）
  │  重点：技术栈、启动流程、并行预取
  ├──► 第三篇：工具系统（必读）
  │      重点：插件化架构、工具注册机制
  ├──► 第四篇：查询引擎（必读）
  │      重点：Agent Loop、流式处理
  ├──► 第六篇：终端 UI（推荐）
  │      重点：React + Ink、虚拟滚动
  ├──► 第十一篇：性能优化（推荐）
  │      重点：懒加载、死代码消除、缓存
  └──► 第七篇：状态管理（选读）
         重点：极简 Store、发布订阅
```

**预计时间**：1-2 周

---

## 路线三：AI 应用开发者

如果你正在构建 AI Agent 系统：

```
第四篇：查询引擎（核心必读）
  │  重点：Agent Loop、Tool Use、Thinking 模式
  ├──► 第十篇：多 Agent 协作（核心必读）
  │      重点：Coordinator、Swarm、Agent 通信
  ├──► 第五篇：权限系统（重要）
  │      重点：AI 安全沙箱、权限分层
  ├──► 第八篇：服务与集成（重要）
  │      重点：MCP 协议、上下文压缩、记忆系统
  └──► 第三篇：工具系统（推荐）
         重点：Tool 接口设计、工具注册
```

**预计时间**：1 周

---

## 路线四：系统架构师

如果你想了解大型项目的架构设计：

```
第二篇：架构全景（全部精读）
  ├──► 第十一篇：性能优化（精读）
  ├──► 第七篇：状态管理（精读）
  ├──► 第九篇：Bridge 桥接（精读）
  └──► 第十二篇：亮点总结（总览）
```

**预计时间**：3-5 天
