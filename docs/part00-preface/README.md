# 前言：关于本书

> 用漫画学技术，让 51 万行源码变得有趣易懂！

---

## 这本书是什么？

**《图解 Claude Code 完全指南》** 是一本零基础友好的技术学习书，用 **哆啦A梦风格的漫画** + **详细源码解析** 的方式，带你彻底理解 Claude Code —— 目前最强大的 AI 编程助手之一。

Claude Code 是 Anthropic 公司开发的命令行 AI 编程代理，它的源码约有 **1,900 个 TypeScript 文件、51 万行代码**。本书将这个庞大的项目拆解成 **12 大主题、120+ 节课程**，每节课都配有：

- 通俗易懂的 **生活类比**（就像用做饭、装修来解释编程概念）
- 生动形象的 **漫画图解**（哆啦A梦风格，轻松有趣）
- 真实的 **源码片段**（不是空谈理论，每行代码都能在源码中找到）
- 直观的 **架构图**（Mermaid 流程图，一图胜千言）
- 实用的 **动手练习**（学以致用，巩固知识）

## 如何使用本书

### 在线阅读（推荐）

直接访问 [GitHub Pages](https://bcefghj.github.io/claude-code-complete-guide) 在线阅读，支持：
- 全文搜索
- 章节导航
- 代码高亮
- 手机适配

### 打印 / PDF

在浏览器中按 `Ctrl+P`（或 `Cmd+P`），选择"另存为 PDF"即可导出当前页面。

### 本地部署

```bash
# 克隆仓库
git clone https://github.com/bcefghj/claude-code-complete-guide.git

# 安装 docsify（如果未安装）
npm i docsify-cli -g

# 启动本地服务
docsify serve docs

# 浏览器访问 http://localhost:3000
```

## 学习建议

1. **顺序阅读**：前 4 篇是基础，建议按顺序阅读
2. **第 5-12 篇可跳读**：根据兴趣选择深入的主题
3. **对照源码**：建议同时打开 Claude Code 源码，对照阅读
4. **做练习**：每节课的动手练习能帮你真正理解概念
5. **多看漫画**：漫画是快速建立直觉的好方法
